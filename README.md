# 🎓 Student Registration System

A simple and responsive Student Registration System built using **Flask**, **HTML**, and **CSS**. This project allows users to register student details through a web form and display the registered students in a table.

---

## 📌 Features

- Student Registration Form
- Displays Registered Students
- Responsive User Interface
- Success Message after Registration
- Student Counter
- Professional Layout using CSS
- Built using Flask Framework

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3

## ▶️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/Student-Registration-System.git
```

### 2. Open the Project Folder

```bash
cd Student-Registration-System
```

### 3. Install Flask

```bash
pip install flask
```

### 4. Run the Application

```bash
python app.py
```

### 5. Open in Browser

```
http://127.0.0.1:5000
```
